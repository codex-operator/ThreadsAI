# ThreadsAI Lite

Облегченная версия расширения без тяжелого парсинга ленты и без системы пресетов промптов.

## Что осталось

- Простое popup-окно с базовыми настройками:
  - модель;
  - лимит комментариев;
  - диапазон задержки.
- Фоновый `service-worker` для сохранения настроек и переключения состояния `start/stop`.
- Минимальный `content-script` для базовой связности сообщений.

## Что удалено

- Парсинг ленты Threads.
- Парсер-модалки и табличный режим.
- Система пресетов/кастомных промптов.
- Дополнительные popup-модули логов и вспомогательных утилит.

## Локальный запуск

1. Откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите корень проекта.
